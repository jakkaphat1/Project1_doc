วิชา โปรเจค 1

สิ่งที่ต้องการทำ : ต้องการทำโปรเจคเกี่ยวกับวิธีตรวจสอบความเหมือนกันของไฟล์ที่นักศึกษาส่งมา เพื่อที่จะจัดกลุ่มว่า กลุ่มที่มีความคล้ายกันมาก มีไฟล์ไหนบ้าง
กลุ่มที่ความคล้ายกันน้อยมีไฟล์ไหนบ้าง โดยที่เราไม่ได้จะต้องจับว่าไฟล์ไหนลอกกัน แต่เราจะบอกว่ากลุ่มนี้มีไฟล์ไหนบ้างที่คล้ายคลึงกัน โดยให้มนุษย์ไปตัดสินใจเอง

Scope ของสิ่งที่ต้องการทำ : ตรวจสอบไฟล์ pdf ที่มี Text และ รูปภาพ โดยมี input เป็น กลุ่มของไฟล์ pdf ที่อาจารย์ส่งลงไปในระบบ 
และ output ออกมาเป็นการจัดกลุ่มของความเหมือนในแต่ละกลุ่ม โดยอาจจะแบ่งเป็น กลุ่มแรกมีความคล้ายกันแบบ > 80% เป็นต้น

สิ่งที่ทำแล้วตอนนี้ : มีการใช้ Text embedding , image embedding มาใช้ในโปรเจค โดยให้ดึงข้อความจากไฟล์ และดึงรูปภาพแยกจากกัน
จากนั้นใช้ embedding model ทั้ง text , image ในการ embedding เป็น vector จากนั้นส่งไปที่ vector database เพื่อเปรียบเทียบความเหมือน



-------------------
สิ่งที่ต้องการทำต่อจากตอนนี้ : 
1.) พัฒนาแนวทางที่ดีกว่านี้ (หากเป็นไปได้) อาจจะมีวิธีการใหม่ เช่น 
	1.ส่วนของ Text ใช้วิธีการ Lexical Matching (n-gram)

	2.ตรวจลอกแบบ rewrite ด้วยการใช้ embedding model + vector database
		-> cosine similarity

	3.ตรวจ metadata 

	4.ส่วนรูปใช้ image embedding + vector database
		-> cosine similarity

แล้วเอามารวมกันเป็น % ตอนสุดท้าย ดีมั้ย

2.) จะมีวิธีการใดในการที่จะทำให้มันสามารถรองรับกับไฟล์จำนวนเยอะ ๆ จากนักศึกษาเกือบ 100 คนได้

3.)  พัฒนาความสามารถในการอธิบายผลลัพธ์ เช่น ระบบบอกได้ว่ากลุ่มนี้มีความเหมือนกันอย่างไรบ้าง...
